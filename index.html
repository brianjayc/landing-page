    FAQ

    Is the windows OS pre-installed?

    Yes, except barebones version.

    What is the maximum memory and speed of B550?

    The maximum memory of the B550 is 64GB —2x32GB stick, and the speed is 3200MHz.

    Is the RAM dual rank or single rank?

    16GB RAM is single rank, and 32GB RAM is dual rank.

    Are there any limitations on the M.2 PCIe expansion bays for SSD cards?

    No, there is no limit, it's a standard PCIE 3.0 x4 SSD.

    How many monitors does B550 support?

    B550 can be connected to 3 monitors at the same time.
